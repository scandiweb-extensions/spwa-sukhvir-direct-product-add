#### Tested on:
* Magento2:`2.4.3`
* ScandiPWA:`5.0.4`
* ScandiPWA Extension: `spwa-sukhvir-direct-product-add:0.0.1`

Features list:

- not available for SPWA v5.0.5 or lower.
